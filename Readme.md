# 🐱 THE CAT

## Descripción

**THE CAT** es una aplicación web interactiva desarrollada con HTML, CSS y JavaScript. El proyecto utiliza imágenes y sonidos para representar un gato animado, permitiendo crear efectos visuales y de interacción mediante programación.

---

## Estructura del Proyecto

```text
THE CAT/
│
├── CSS/
│   └── style.css
│
├── scripts/
│   └── main.js
│
├── src/
│   ├── Fondo(Completo).png
│   ├── Gato_Cabeza.png
│   ├── Gato(Cuerpo).png
│   ├── Gato(Cola).png
│   ├── Mau1.mp3
│   ├── Mau2.mp3
│   ├── MaullidoFrame1.png
│   └── MaullidoFrame2.png
│
└── index.html
```

---

##  Objetivo

Desarrollar una animación interactiva de un gato utilizando recursos multimedia (imágenes y sonidos) para aplicar conceptos de diseño web, estilos CSS y programación con JavaScript.

---

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Imágenes PNG
- Audio MP3

---

## 📄 Descripción de los Archivos

### index.html
Contiene la estructura principal de la página web.

### CSS/style.css
Define los estilos visuales, animaciones y posicionamiento de los elementos.

### scripts/main.js
Implementa la lógica del programa, eventos e interacciones del usuario.

### Carpeta src

| Archivo | Función |
|----------|----------|
| Fondo(Completo).png | Imagen de fondo de la escena |
| Gato_Cabeza.png | Cabeza del gato |
| Gato(Cuerpo).png | Cuerpo del gato |
| Gato(Cola).png | Cola del gato |
| Mau1.mp3 | Sonido de maullido 1 |
| Mau2.mp3 | Sonido de maullido 2 |
| MaullidoFrame1.png | Primer frame de la animación de maullido |
| MaullidoFrame2.png | Segundo frame de la animación de maullido |

---

##  Cómo Ejecutar el Proyecto

1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar el archivo `index.html`.
4. También se recomienda utilizar la extensión **Live Server** para una mejor experiencia de desarrollo.

---

##  Funcionalidades

- Visualización de un gato mediante imágenes superpuestas.
- Reproducción de sonidos de maullido.
- Animaciones creadas con CSS y JavaScript.
- Interacción con eventos del usuario.
- Organización modular de recursos multimedia.
